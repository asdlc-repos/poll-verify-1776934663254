# Overview

A basic todo list application where people can add tasks, mark them as done, and remove them from the list.

# Personas

- User — adds and manages their personal list of tasks.

# Features

- Add a new task by typing it in and hitting enter.
- See all tasks in a simple list.
- Mark a task as complete by clicking a checkbox.
- Delete a task from the list.
- See which tasks are done and which are still pending.
